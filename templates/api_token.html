API使用说明

上传图片

使用以下命令上传图片：

curl -X POST "http://localhost:8000/upload" -F "file=@图片路径.jpg" -F "token={{ user.api_token }}"

查看图片列表

获取您上传的所有图片：

curl "http://localhost:8000/images?user_id={{ user.id }}"
